با نصب این پلاگین می توانید از خدمات درگاه واسط و مستقیم آل‌سات پرداخت برروی اسکریپت فروش فایل easy digital downloads استفاده کنید!

== Description ==
 افزونه edd-alsatpardakht-gateway امکان فروش اینترنتی و آنلاین فایل را از طریق درگاه آل‌سات پرداخت به افزونه اضافه می کند.
برای فعالسازی درگاه واسط از بخش تنظیمات پلاگین استفاده از IPG واسط را فعال کنید.


== Installation ==
برای فعال سازی درگاه آل‌سات پرداخت مراحل زیر را دنبال کنید:

1. در سایت https://alsatpardakht.com ثبت نام کنید و درگاه ایجاد کنید.
2. پلاگین را نصب و فعال کنید.
3. به قسمت تنظیمات (پیکربندی) پلاگین و بخش درگاه های پرداخت مراجعه کنید.
4. درگاه آل‌سات پرداخت را فعال و در قسمت کد دریافتی از پنل (API) مقدار API درگاه خود را وارد کنید.
5. درآمد بیشتری کسب کنین.
